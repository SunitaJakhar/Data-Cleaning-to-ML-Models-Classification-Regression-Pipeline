# Data-Cleaning-to-ML-Models-Classification-Regression-Pipeline

From Raw Data to Predictive Models: A Machine Learning Pipeline for Classification and Regression Analysis

This project implemented a complete data science pipeline across three tasks, demonstrating proficiency in data quality management, information-theoretic analysis, and predictive modelling. The workflow followed the canonical sequence described in the machine learning literature: data inspection, cleaning, exploration, and modelling — with each stage informing the next.

Task 1 established the importance of addressing data quality issues (missing labels, outliers) before any analytical inference. 

Task 2 grounded the classification problem in probabilistic and information-theoretic foundations — Gini impurity, entropy, and misclassification rate — which provide both diagnostic value and theoretical motivation for algorithm selection. 

Task 3 demonstrated the process of choosing, fitting, and evaluating a regression model based on visual trend identification, yielding a polynomial model that accurately captures the quadratic structure of the time-series data.

Collectively, the three tasks illustrate that effective data science is not merely the application of algorithms, but a principled process of understanding data, making justified analytical decisions, and critically evaluating outputs at each stage.
